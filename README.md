## Technologies

* Visual Studio 2019
* .NET Framework 4.6.1
* Windows Forms App

## Setup

1. Clone this repository by using git
```
git clone https://github.com/Zacky3181V/prog_lect_wfa
```
2. Open in in Visual Studio
3. Before running choose which project you would like to Run
4. Run!
